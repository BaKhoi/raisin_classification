## Requirements
- Python 3.9+
- `numpy==1.26.4`
- `scikit-learn==1.3.1`
- `pickle`
- `matplotlib==3.8.4`
- `pandas==2.2.1`
- `matplotlib==3.8.4`

## Installation
1. Clone the repository
```
git clone https://github.com/BaKhoi/raisin_classification.git

cd raisin_classification

```

2. Install required packages
```
pip install numpy scikit-learn pickle
```

## Usage
1. Run the script
```
python main.py
```
